Framing the Problem

  •  Alejandro Rodríguez
  • 12 Comentarios

¿Qué es?

Framing the problem, es una herramienta que te ayudará a delimitar claramente el problema al que le quieres dar solución. Funciona por medio de preguntas clave que te guían hasta el aterrizaje del concepto de la problemática.

Pasos:

1. Identifica claramente los elementos del problema que quieres resolver. ¿Quién tiene el problema? + Causa + Efecto. Luis + comió dulces + le duele la panza.

2. Crea una pregunta con los elementos del problema: ¿Cómo podría reducir el dolor de panza de Luis debido a que comió muchos dulces?

3. No incluyas una solución en la pregunta, a no ser que sea una restricción.

¿Cómo podría reducir el dolor de panza de Luis con medicamentos, debido a que comió muchos dulces? (ya estás estableciendo que lo tienes que hacer con medicamentos, si no es una restricción, no lo incluyas, ya que el problema se puede resolver de diferentes maneras).

4. Parafrasea la pregunta, utilizando diferentes palabras sin cambiar el significado original.

  • ¿Cómo podría reducir el dolor de panza en niños cuando comen muchos dulces?
  • ¿Qué remedios existen para reducir el dolor de panza para niños cuando comen muchos dulces?
  • ¿Cómo puedo evitar que los niños que comen muchos dulces les duela la panza?
  • ¿Cómo podría reducir la cantidad de dulces que comen los niños, para que no les duela la panza?

Realiza este ejercicio hasta que encuentres la opción que más adecuada para darle respuesta.

¿Cuándo usar la herramienta?

Cuando se tiene la necesidad de aterrizar y comprender en un concepto una problemática.
